# Score_Prediction using Machine Learning

## Objective

Predict student scores based on study hours.

## Dataset

Dataset containing study hours and corresponding scores.

## Tools Used

* Python (Pandas, NumPy)
* Visualization (Matplotlib)
* Machine Learning (Scikit-learn)

## Analysis Performed

* Data cleaning
* Correlation analysis

## Models Used

* Linear Regression

## Results

* Model predicts scores with good accuracy

## Key Insights

* Strong positive correlation between study hours and scores

## Future Work

* Extend with more features for better predictions
